# Claude History Visualizer

Visualize your Claude Code `history.jsonl` — word clouds, connection graphs, emotion trends, timeline heatmaps, and more.

**All processing happens in your browser. Nothing is uploaded.**

## Usage

1. Open `index.html` (or host it anywhere)
2. Drop your `~/.claude/history.jsonl` file
3. Explore the visualizations

## Visualizations

- **Word Cloud** — most used terms
- **Word Graph** — D3 force-directed co-occurrence graph
- **Emotions** — sentiment analysis with emoji breakdown
- **Timeline** — activity heatmap, hourly/daily charts
- **Projects** — distribution across projects
- **Particle Flow** — animated word particles with connections
